## 核心公式速查

| 概念 | 公式 |
|------|------|
| 效用最大化 | max U(x,y) s.t. px·x + py·y = I |
| 拉格朗日条件 | MUx/px = MUy/py = λ |
| 需求弹性 | ε = (ΔQ/Q) / (ΔP/P) |
| 利润最大化 | MR = MC |
| 完全竞争 | P = MC（短期）；P = min AC（长期） |
| 垄断加价 | (P - MC) / P = 1 / \|ε\| |
| 纳什均衡 | 给定对方策略，己方无法单方面改善 |
| 现值 | PV = FV / (1+r)^t |
| 期望效用 | EU = Σ pᵢ · U(xᵢ) |
