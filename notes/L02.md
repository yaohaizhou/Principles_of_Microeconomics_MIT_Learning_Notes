## 核心问题

> 如何用数学描述人的偏好？

---

## 关键概念

| 概念 | 定义 | 备注 |
|------|------|------|
| 完备性 (Completeness) | 消费者对任何商品组合都能明确排序或表示无差异，不能说"不知道" | 偏好公理之一 |
| 传递性 (Transitivity) | A>B 且 B>C → A>C；偏好不能循环矛盾 | 偏好公理之一 |
| 非餍足性 (Nonsatiation) | 越多越好，永远认为商品数量多更好 | "More is better" |
| 效用函数 U(x,y) | 将消费组合映射为数字，用于排序偏好；数值是序数，无绝对含义 | Ordinal，非 Cardinal |
| 无差异曲线 | 给消费者带来相同效用的所有消费组合的连线 | 向下倾斜、凸向原点、不相交 |
| 边际效用 MU | 多消费一单位商品带来的额外效用；具有递减性 | MU = dU/dQ |
| 边际替代率 MRS | 无差异曲线斜率；为多获得一单位X愿意放弃的Y的数量 | MRS = -MUx/MUy |

---

## 重要公式 / 模型

```
MRS = -ΔY/ΔX = MUx / MUy

边际替代率递减：沿无差异曲线移动，X增多 → MUx↓，Y减少 → MUy↑ → MRS递减
```

无差异曲线的四个特征：
1. 越高（离原点越远）效用越大
2. 向下倾斜（非餍足性要求）
3. 永不相交（**传递性**要求，见下方证明）
4. 凸向原点（MRS递减）

**为什么无差异曲线不能相交（反证法）：**

假设 IC₁ 和 IC₂ 相交于点 P，在 IC₁ 上另取点 A，在 IC₂ 上另取点 B，且 B 的两种商品都严格多于 A：

```
由 IC₁：U(A) = U(P)
由 IC₂：U(B) = U(P)
由传递性：U(A) = U(B)

但 B 的 x 和 y 都多于 A
由非餍足性：U(B) > U(A)  → 矛盾！
```

> 注意：直接违反的是**传递性**；非餍足性是完成最后一步推导的辅助工具。两者缺一不可。

---

## 直觉理解

无差异曲线 ≈ 机器学习里的等高线（loss contour）：同一条曲线上的所有点，"loss"（负效用）相同。效用最大化就是找到最高的那条等高线上能到达的点。

MRS递减 ≈ 正则化的直觉：你已经有很多X了，再多一点X的价值就越来越小，所以你不愿意为它付出太多Y。

---

## 现实应用

- **星巴克大杯 vs 中杯定价**：大杯容量是中杯的1.5倍，但价格远不到1.5倍——因为边际效用递减，消费者愿意为额外饮料支付的意愿在降低，商家基于此制定阶梯定价
- **效用是序数不是基数**：不能说"我今天的幸福感比昨天高30%"，只能说"我更喜欢今天的状态"

**A**：

**A**：效用是**序数 (Ordinal)**，只用来排序选择，数值本身无意义。正因如此，效用函数可以做任意单调变换（如 U² 或 ln U）而不改变任何结论——这大大简化了分析，选取最方便计算的函数形式即可。

## 练习题

**题 1**：
效用函数 U(x,y) = x·y，求在消费点 (2, 4) 处的 MRS。

**解**：
MUx = ∂U/∂x = y = 4
MUy = ∂U/∂y = x = 2
MRS = -MUx/MUy = -4/2 = **-2**

含义：在该点，消费者愿意放弃2单位Y来换取1单位X（保持效用不变）。

---

## 最优消费条件：MRS = Px/Py

消费者在 MRS ≠ 价格比时会主动调整消费，直到两者相等。

**直觉解释（以猪肉为例）：**

- MRS = 3 意味着消费者愿意付 3 块买 1 斤猪肉（主观估值）
- Px/Py = 2 意味着市场上 1 斤猪肉只要 2 块（市场要价）
- 主观估值 > 市场价 → 每买一斤都"赚到"1块的主观价值 → 继续买

由于边际效用递减，猪肉买得越多，主观估值越低：

```
第1斤：愿意付3块
第2斤：愿意付2.5块
第3斤：愿意付2块  ← 停！MRS = Px/Py
第4斤：愿意付1.5块（不买了）
```

**三种状态：**

| 状态 | 含义 | 行动 |
|------|------|------|
| MRS > Px/Py | 主观估值 > 市场价，X 被低估 | 买更多 X |
| MRS = Px/Py | 主观估值 = 市场价 | 最优，停止 |
| MRS < Px/Py | 主观估值 < 市场价，X 买多了 | 减少 X |

**最优消费条件本质：你对商品的主观估值，恰好等于市场的要价。** L03 会用预算约束正式推导这个条件。

## 串联总结：从边际效用到市场价格

这几节课的核心逻辑可以串成一条完整链条：

```text
边际效用
→ 决定个人愿付价格（最高愿付价格）
→ 很多个体的愿付价格汇总成需求曲线
→ 需求曲线与供给共同决定市场价格
→ 市场价格低于部分人的愿付价格时，产生消费者剩余
→ 拍卖机制只是把这种竞争过程显性化
```

### 1. 边际效用决定个人愿付价格

对每个消费者来说，都存在一个**最高愿付价格**：
- 市场价格低于它 → 愿意买
- 市场价格高于它 → 不买

本质上，商品带来的边际效用越高，消费者愿意放弃的其他资源越多；如果用货币来衡量，这就表现为更高的愿付价格。

### 2. 很多个体的愿付价格汇总成需求曲线

假设五个人对一张票的最高愿付价格分别为：
- 2000
- 1800
- 1500
- 1200
- 500

那么：
- 价格为 1800 时，只有前两个人买
- 价格为 1500 时，前三个人买
- 价格为 1200 时，前四个人买

把"价格"和"愿意购买的人数"对应起来，就是**需求曲线**。所以需求不是凭空存在的，而是许多个体主观估值的汇总。

### 3. 市场价格由边际买家决定

如果市场上只有 3 张票，最终只有愿付价格最高的前三个人能买到。此时市场价格会落在第 3 名（1500）和第 4 名（1200）之间。

也就是说，价格不是由"最想买的人"单独决定，而是由**边际上那个买得到的人**和**边际上那个买不到的人**共同夹出来的。

### 4. 消费者剩余

如果最终市场价格是 1300：
- 愿付 2000 的人：消费者剩余 = 700
- 愿付 1800 的人：消费者剩余 = 500
- 愿付 1500 的人：消费者剩余 = 200

所以：

```text
消费者剩余 = 愿付价格 - 实际支付价格
```

它反映的是消费者从交易中获得的净收益。

### 5. 拍卖机制：把竞争过程显性化

拍卖揭示了一个非常关键的区别：
- **边际效用决定个人愿付价格**
- **市场竞争决定最终成交价格**

例如一件古董只有一件：
- A 最多愿付 100 万
- B 最多愿付 80 万
- C 最多愿付 50 万

最后 A 会买到，但成交价通常不会是 100 万，而会接近第二高出价者的水平（约 80 万）。

所以赢家通常不会付到自己的心理上限，他仍然可以获得消费者剩余。

### 6. 用古董统一理解

一件稀缺古董为什么贵？
1. **供给极少**：只有一件
2. **部分买家的边际效用极高**：他们非常想要
3. **高估值买家之间发生竞争**
4. **价格被抬到接近次高估值者的水平**

因此，古董价格高并不只是因为"稀缺"，而是：

> **稀缺供给 + 高边际效用买家的竞争**
