## 核心问题

> 如何用数学描述人的偏好？

---

## 关键概念

| 概念 | 定义 | 备注 |
|------|------|------|
| 完备性 (Completeness) | 消费者对任何商品组合都能明确排序或表示无差异，不能说"不知道" | 偏好公理之一 |
| 传递性 (Transitivity) | A>B 且 B>C → A>C；偏好不能循环矛盾 | 偏好公理之一 |
| 非餍足性 (Nonsatiation) | 越多越好，永远认为商品数量多更好 | "More is better" |
| 效用函数 U(x,y) | 将消费组合映射为数字，用于排序偏好；数值是序数，无绝对含义 | Ordinal，非 Cardinal |
| 无差异曲线 | 给消费者带来相同效用的所有消费组合的连线 | 向下倾斜、凸向原点、不相交 |
| 边际效用 MU | 多消费一单位商品带来的额外效用；具有递减性 | MU = dU/dQ |
| 边际替代率 MRS | 无差异曲线斜率；为多获得一单位X愿意放弃的Y的数量 | MRS = -MUx/MUy |

---

## 重要公式 / 模型

```
MRS = -ΔY/ΔX = MUx / MUy

边际替代率递减：沿无差异曲线移动，X增多 → MUx↓，Y减少 → MUy↑ → MRS递减
```

无差异曲线的四个特征：
1. 越高（离原点越远）效用越大
2. 向下倾斜（非餍足性要求）
3. 永不相交（传递性要求）
4. 凸向原点（MRS递减）

---

## 直觉理解

无差异曲线 ≈ 机器学习里的等高线（loss contour）：同一条曲线上的所有点，"loss"（负效用）相同。效用最大化就是找到最高的那条等高线上能到达的点。

MRS递减 ≈ 正则化的直觉：你已经有很多X了，再多一点X的价值就越来越小，所以你不愿意为它付出太多Y。

---

## 现实应用

- **星巴克大杯 vs 中杯定价**：大杯容量是中杯的1.5倍，但价格远不到1.5倍——因为边际效用递减，消费者愿意为额外饮料支付的意愿在降低，商家基于此制定阶梯定价
- **效用是序数不是基数**：不能说"我今天的幸福感比昨天高30%"，只能说"我更喜欢今天的状态"

## NotebookLM 问答记录

**Q**：这节课最重要的3个概念及关键图形？
**A**：
1. **偏好三公理**：完备性、传递性、非餍足性——这是整个消费者理论的数学地基
2. **无差异曲线**：向下倾斜、不相交、凸向原点；四条性质每条都对应一个偏好假设
3. **边际效用递减 → MRS递减**：从数学上解释了无差异曲线为何凸向原点

**Q**：效用是序数还是基数，为什么这个区别重要？
**A**：效用是**序数 (Ordinal)**，只用来排序选择，数值本身无意义。正因如此，效用函数可以做任意单调变换（如 U² 或 ln U）而不改变任何结论——这大大简化了分析，选取最方便计算的函数形式即可。

## 练习题

**题 1**：
效用函数 U(x,y) = x·y，求在消费点 (2, 4) 处的 MRS。

**解**：
MUx = ∂U/∂x = y = 4
MUy = ∂U/∂y = x = 2
MRS = -MUx/MUy = -4/2 = **-2**

含义：在该点，消费者愿意放弃2单位Y来换取1单位X（保持效用不变）。
