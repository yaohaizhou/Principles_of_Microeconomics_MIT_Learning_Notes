## 核心问题

> **理论核心**：价格变化如何影响消费量？如何将价格变化的效果分解为收入效应和替代效应？
>
> **逻辑链完成**：L01给出需求曲线 → L02建立偏好与效用 → L03引入预算约束与受约束优化 → **L04从个体最优选择推导出需求曲线，并揭示价格变化背后的双重机制**

---

## 关键概念

| 概念 | 英文 | 定义与含义 |
|------|------|------|
| 需求曲线推导 | Demand Curve Derivation | 需求曲线不是凭空出现的，而是消费者在不同价格下进行受约束优化（效用最大化）的结果。通过改变一种商品价格、重新求切点、连接价格-数量坐标点，即可推导出向下倾斜的需求曲线 |
| 替代效应 | Substitution Effect | 在保持消费者效用（幸福感）不变的前提下，纯粹由商品相对价格变化引起的消费量变化。又称"补偿性需求"（Compensated Demand）。**符号永远为负**：价格上升 → 需求量必然下降 |
| 收入效应 | Income Effect | 在保持相对价格比不变的前提下，纯粹由价格变化导致实际购买力（有效收入）变化所引起的消费量变化。**方向不确定**：取决于商品是正常商品还是劣等商品 |
| 正常商品 | Normal Good | 收入增加时需求量增加的商品。收入效应方向与替代效应一致（同为负），两者加强 |
| 劣等商品 | Inferior Good | 收入增加时需求量反而减少的商品（如快餐、土豆）。收入效应方向与替代效应相反，两者对抗 |
| 吉芬商品 | Giffen Good | 极端罕见的商品，需求曲线向上倾斜（价格越高反而买越多）。条件：必须是劣等商品，且在预算中占极大比例，使正向收入效应完全压倒负向替代效应 |
| 需求弹性 | Demand Elasticity | 需求量对价格变化的敏感程度。完全无弹性（垂直曲线，如胰岛素）到完全有弹性（水平曲线，如口香糖品牌）|

## 重要公式 / 模型

### 1. 需求曲线推导过程

**步骤**：
```
1. 给定收入Y、商品A价格Pa、商品B价格Pb
   → 画出预算约束线，找到与无差异曲线的切点（最优消费组合）

2. 改变Pb（如从$6→$9→$4）
   → 预算线以A截距为轴旋转，找到新的切点

3. 提取每个价格下的最优消费量
   → (P=$9, Q=4), (P=$6, Q=6), (P=$4, Q=9)

4. 在P-Q坐标系中连线 → 需求曲线
```

### 2. 收入效应与替代效应分解

**图形分解法**：
```
价格上涨时（如 Pb: $1 → $3）：

总效应 = 替代效应 + 收入效应

替代效应：点a → 点b（沿原无差异曲线滑动到新价格比切线）
  → 画一条具有新斜率、但与原无差异曲线相切的虚拟预算线
  → 保持效用不变，仅反映相对价格变化

收入效应：点b → 点c（从虚拟预算线平移到真实预算线）
  → 保持新价格比不变，仅反映购买力变化
```

### 3. 效应方向总结

```
                    替代效应    收入效应    总效应
正常商品 (Normal)：   负         负         负（需求曲线向下）
劣等商品 (Inferior)： 负         正         通常为负（替代效应 > 收入效应）
吉芬商品 (Giffen)：   负         正(极大)   正（需求曲线向上！）
```

## 直觉理解

**需求曲线 = 个体最优选择的"连续快照"**：
- 想象你面对不同的饼干价格，每次都重新做最优选择
- 把每次"价格→最优数量"的结果画出来，就是需求曲线
- 需求曲线不是假设出来的，而是效用最大化的自然推论

**替代效应 = "性价比驱动的切换"**：
- 价格涨了，性价比变差了，你自然会转向替代品
- 这是在"保持幸福感不变"的假想补偿下的纯理性切换
- **永远是负的**：变贵 → 少买，这是不可违背的逻辑

**收入效应 = "变穷/变富后的行为变化"**：
- 价格涨了 → 你实质上变穷了（购买力下降）
- 变穷后你会少买奢侈品（正常商品），但可能多买便宜货（劣等商品）
- 方向取决于商品类型，这就是有趣的地方

**类比：替代效应 ≈ A/B Testing 中的"控制变量"**
- 替代效应：控制"幸福感"不变，只看价格变化的影响 → 类似 A/B test 中控制其他变量
- 收入效应：控制"价格比"不变，只看购买力变化的影响
- 两步分解法 = 经济学的"控制变量实验"

## 现实应用

### 应用一：需求弹性的极端案例 — 胰岛素 vs 口香糖

**完全无弹性 — 胰岛素**：
- 糖尿病患者必需，没有可行替代品（no plausible substitute）
- 无论价格如何上涨，购买量不变 → 需求曲线垂直
- 类似案例：水、污水处理

**极富弹性 — 口香糖品牌**：
- 不同品牌口香糖（Orbit vs Eclipse）之间几乎完美替代
- 某品牌哪怕只贵一美分，消费者立刻转向竞品
- 类似案例：仿制药、快餐品牌

### 应用二：劣等商品与正常商品的现实表现

**劣等商品 — 快餐与土豆**：
- 快餐（如麦当劳）：你越有钱反而越不想吃
- 旧时代的土豆：穷人用来填饱肚子的便宜食物，有钱后转向牛排

**正常商品与奢侈品 — 手表与游艇**：
- 劳力士、游艇：收入增加不仅买得更多，预算占比也更大

### 应用三：中国大米实验 — 吉芬商品的实证验证

**实验背景**：研究人员在中国极度贫困地区给家庭发放大米折扣券（实质降低大米价格）

**中度贫困家庭**：大米降价 → 买更多大米（正常的向下倾斜需求曲线）

**极度贫困家庭（吉芬现象）**：大米降价 → **反而减少购买大米**
- 原因：大米占据预算的绝大部分，降价相当于"变富"
- 变富后不再需要顿顿吃大米，用省下的钱买蔬菜和肉
- 即：巨大的正向收入效应完全压倒了负向替代效应

**意义**：这是经济学史上首次用严格实验证实了吉芬商品在现实中的存在

### 应用四：牛排与土豆 — 双效应对抗

土豆价格从 $1 涨到 $3：
- **替代效应**：土豆变贵 → 少吃土豆多吃牛排（需求下降）
- **收入效应**：土豆涨价 → 实质变穷 → 土豆是劣等商品 → 变穷反而多吃土豆（需求上升）
- 两个效应方向相反，互相"打架"，这正是劣等商品分析的精髓

## 练习题

**题 1**：从消费者选择理论推导需求曲线

假设消费者总预算为 $72，披萨价格固定为 $12。在特定效用函数下，无论饼干价格如何变化，消费者优化后总选择 3 块披萨。已知：
- 情境A：饼干 $6 → 购买 6 块饼干
- 情境B：饼干 $9 → 购买 4 块饼干
- 情境C：饼干 $4 → 购买 9 块饼干

问题：
1. 三种情境下预算约束线的截距和斜率如何变化？
2. 列出饼干需求曲线的坐标点，并解释为何向下倾斜。

**解**：
1. **Y轴截距（披萨）**：始终为 72/12 = 6 块（收入和披萨价格不变）

   **X轴截距与斜率**：
   - 情境A：X截距 = 72/6 = 12，斜率绝对值 = 6/12 = 1/2
   - 情境B：X截距 = 72/9 = 8（向内收缩），斜率绝对值 = 9/12 = 3/4（更陡）
   - 情境C：X截距 = 72/4 = 18（向外扩张），斜率绝对值 = 4/12 = 1/3（更平）

2. **需求曲线坐标点**：(P=9, Q=4)、(P=6, Q=6)、(P=4, Q=9)

   **向下倾斜的原因**：价格上升时，预算线以Y轴截距为轴向内旋转，消费者在更小的机会集中重新优化，新切点对应更少的饼干消费量。需求曲线是受约束优化的自然推论，不是假设。

**题 2**：劣等商品的收入效应与替代效应分解

消费者面对牛排（$5）和土豆（初始 $1），预算 $25。初始最优：3.5 块牛排 + 7.5 个土豆。土豆涨到 $3 后：
- 最终状态（点c）：5 个土豆
- 补偿性需求状态（点b，保持原效用）：4 个土豆

问题：
1. 计算总效应、替代效应和收入效应。
2. 为什么收入效应方向与替代效应相反？

**解**：
1. **总效应** = 5 - 7.5 = **-2.5 个土豆**
   **替代效应** = 4 - 7.5 = **-3.5 个土豆**（沿原无差异曲线，a→b）
   **收入效应** = 5 - 4 = **+1 个土豆**（平移预算线，b→c）
   验证：-3.5 + 1 = -2.5 ✓

2. **替代效应为负**：土豆变贵，性价比下降，消费者沿无差异曲线减少土豆、增加牛排
   **收入效应为正**：土豆涨价使购买力下降（实质变穷）。因为土豆是劣等商品，变穷时反而需要多买土豆来填饱肚子。两效应对抗，但替代效应更大，总需求仍然下降
